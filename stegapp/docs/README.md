## 🎯 Projet StegApp

Application de stéganographie web permettant de cacher et extraire des données dans :
- **Images** : PNG, JPG, JPEG, BMP, TIFF
- **Audio** : WAV PCM 16-bit
- **PDF** : Métadonnées

## 🚀 Démarrage rapide

```bash
cd stegapp/backend
python -m venv venv
venv/Scripts/activate
pip install -r requirements.txt
python api.py
```
